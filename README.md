# 📊 Flipkart Customer Support Analysis & Machine Learning

## 📌 Project Overview

This project analyzes Flipkart's customer support data using **Exploratory Data Analysis (EDA)** and **Machine Learning (ML)** techniques. The goal is to identify customer support trends, understand factors affecting customer satisfaction (CSAT), and build a predictive model to support better business decisions.

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing.
- Analyze customer support trends using EDA.
- Visualize key insights with charts and graphs.
- Identify factors affecting Customer Satisfaction (CSAT).
- Build and evaluate a Machine Learning model.
- Generate business recommendations based on the analysis.

## 📊 Key Insights

- Customer satisfaction varies across different support categories and channels.
- Some product categories receive significantly more support requests.
- Agent tenure and handling time influence customer satisfaction.
- Data visualization helps identify improvement opportunities in customer support operations.

---

## 💼 Business Recommendations

- Improve support quality for categories with lower CSAT scores.
- Reduce customer response and handling time.
- Provide additional training to customer support agents.
- Optimize staffing during peak support hours.
- Focus on improving products with higher complaint volumes.
